# DearCyFi
DearCyFi is a specialized financial charting widget for use in the DearCyGui GUI library.

## Install

Editable install for development:

```bash
python -m pip install -e .
```

Standard install from source tree:

```bash
python -m pip install .
```

## Quick Import Check

```python
from dearcyfi import DearCyFi
```

## Migration Note

Canonical import path is now:

```python
from dearcyfi import DearCyFi
```

Legacy import `from DearCyFi import DearCyFi` is provided as a temporary compatibility shim for one transition release.
